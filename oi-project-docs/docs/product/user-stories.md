# User Stories

## Device control

As a user, I can say "mute yourself for 30 minutes" and the device will stop requesting attention until the timer expires.

## Status

As a user, I can ask "what are you doing?" and get a short list of active tasks.

## Capture

As a user, I can long-hold and say an idea while walking, then have it saved to my wiki inbox.

## Coding assessment

As a developer, I can ask the agent to assess whether a repo change is viable, and it will inspect read-only context before proposing any edits.

## Safe repo write

As a developer, I can approve a patch from a richer screen after reviewing a diff, while the Stick only asks for compact approval.

## Multi-device routing

As a user, I can ask through the Stick and receive the long answer on my desktop.

## Character status

As a user, I can glance at the device character and know whether the agent is listening, thinking, blocked, muted, or offline.

## Away from home

As a user, I can talk to the agent through a phone relay or VPN while away, while the home server remains the primary runtime.

## Home automation

As a user, I can ask "make the room quiet for recording" and the agent can dim lights, pause media, and set status, after applying safety policy.

## Privacy

As a user, I can ask "what can hear me right now?" and get a truthful answer.
