# Product Positioning

## One-liner

A local-first personal agent OS for tiny voice terminals and every device you already own.

## For technical users

Agent Stick is a hackable embodied interface to a persistent local agent that can manage tasks, tools, memory, and devices without trapping you in a phone or cloud assistant.

## Against voice assistants

Voice assistants are command appliances. Agent Stick is a personal control plane.

## Against chatbots

Chatbots are sessions. Agent Stick is an ongoing operational partner with devices, memory, permissions, and task state.

## Against automation tools

Automation tools need you to predefine flows. Agent Stick can reason, delegate, and ask for approval.

## Wedge

Start with people who already run:

- Home Assistant;
- Raspberry Pi;
- local LLMs;
- dev servers;
- self-hosted services;
- automation scripts.

They understand the pain and will tolerate early rough edges.

## Emotional pitch

A tiny creature on your desk knows what you are working on, but does not interrupt you. You talk to it when you want. It does the boring parts, asks before dangerous parts, and leaves receipts.

## Practical pitch

Voice capture + local server + persistent agent + safe tool broker + multi-device routing.
